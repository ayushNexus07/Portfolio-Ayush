# Ayush Kumar Gupta - Personal Portfolio

A modern, responsive, and interactive personal portfolio website designed to showcase my skills, projects, and journey as a Data Analytics student and Frontend Developer. Built with a focus on premium aesthetics, glassmorphism UI, and smooth animations.

## 🚀 Features

- **Modern UI/UX**: Premium aesthetic with a dark mode glassmorphism design.
- **Interactive Hero Section**: Dynamic particle background canvas.
- **Responsive Design**: Fully adaptable to all screen sizes (mobile, tablet, desktop).
- **Smooth Animations**: Reveal on scroll animations and micro-interactions.
- **Project Showcase**: Dedicated section to display featured technical projects.
- **Timeline**: Visual representation of education and achievements.
- **Contact Form**: Functional UI for getting in touch.

## 🛠️ Technologies Used

- **HTML5**: Semantic and accessible markup.
- **CSS3**: Custom styling, CSS variables, Flexbox/Grid, and animations.
- **JavaScript (Vanilla)**: DOM manipulation, scroll events, modal logic, and canvas animations.
- **FontAwesome**: Scalable vector icons.
- **Google Fonts**: Custom typography (Outfit font family).

## 📂 Project Structure

```
├── index.html   # Main HTML document
├── style.css    # Custom CSS styles, variables, and animations
├── script.js    # JavaScript logic for DOM interactions and canvas
└── README.md    # Project documentation
```

## 💻 Getting Started

To view the project locally:

1. Clone this repository or download the source code files.
2. Navigate to the project directory.
3. Open `index.html` in your preferred web browser. No local server or build tools required.
